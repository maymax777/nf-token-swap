# Basic NFTSwap Project

This project demonstrates a basic Hardhat use case. 

Try running some of the following tasks:

## install packages
   npm install
## deploy
   npm run deploy
## test
   npm run test

## verify
 
 npx hardhat verify --network ropsten --constructor-args arguments.js CONTRACT_ADDRESS
